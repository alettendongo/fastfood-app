# 🍔 Fastfood App – Application de commande en ligne

---

## 🎯 Description

Fastfood App est une application web permettant aux utilisateurs de consulter un menu et de passer des commandes en ligne.

Le projet est composé de deux parties :
- 🔧 Backend : Laravel (API)
- 🎨 Frontend : React

---

## 🧱 Architecture du projet

- Backend : Laravel (API REST)
- Frontend : React JS
- Base de données : MySQL
- Déploiement frontend : Vercel

---

## 💻 Technologies utilisées

### Backend
- Laravel
- PHP
- MySQL

### Frontend
- React JS
- JavaScript
- HTML / CSS

---

## 🚀 Fonctionnalités

- Affichage du restaurant
- Affichage du menu
- Ajout au panier
- Système de commande
- Interface utilisateur moderne
